# 📌 Sobre o Projeto

Repositório destinado ao compartilhamento do projeto e dos artefatos desenvolvidos durante o programa de aceleração.

O programa tem como objetivo desenvolver o senso crítico na escolha da melhor ferramenta de Inteligência Artificial para cada contexto de negócio, considerando diferentes cenários de aplicação, especialmente nas etapas de tratamento e análise de dados.

---

# 🎯 Objetivo

Demonstrar boas práticas no uso de IA aplicada ao tratamento de dados, buscando:

- Avaliar qual ferramenta de IA é mais adequada para cada necessidade;
- Utilizar amostragem da base de dados para otimizar processamento;
- Criar arquivos de documentação (`schema.md`) para fornecer contexto estruturado às IAs;
- Orientar a IA na execução da limpeza e organização dos dados;
- Reduzir o consumo de tokens por meio de estratégias mais eficientes;
- Solicitar à IA a geração de scripts em Python para execução em bases completas, quando necessário.

---

# 📊 Tratamento de Dados

Foi utilizada uma planilha Excel contendo colunas com dados desorganizados e inconsistentes.

O processo de tratamento e limpeza dos dados foi realizado com apoio do **Codex AI**, utilizando instruções estruturadas para garantir maior eficiência e padronização dos resultados.

---

# 📈 Desenvolvimento do Dashboard

Foram desenvolvidas três versões do dashboard, utilizando diferentes ferramentas de IA para fins comparativos:

- GPT Plus
- Claude Sonnet 4.6
- Gemini

O objetivo foi analisar a capacidade de cada ferramenta na construção de dashboards, geração de insights e apoio ao desenvolvimento visual e analítico do projeto.
